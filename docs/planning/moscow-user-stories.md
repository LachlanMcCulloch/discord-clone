## **Must Have** (MVP - Core Discord Experience)

**Authentication & Users**

1. As a user, I can sign up with email/password so I can create an account
2. As a user, I can log in/out so I can access my workspace securely
3. As a user, I can see my profile (username, avatar) so I can identify myself

**Servers & Channels** 4. As a user, I can create a server so I can organize conversations 5. As a user, I can join a server via invite link so I can participate in communities 6. As a user, I can see a list of servers I'm a member of 7. As a user, I can create text channels within a server so I can organize topics 8. As a user, I can see all channels in a server I'm in

**Messaging** 9. As a user, I can send text messages in a channel so I can communicate 10. As a user, I can see messages in real-time without refreshing so conversations feel live 11. As a user, I can see message history when I join a channel 12. As a user, I can see who sent each message and when

**Presence** 13. As a user, I can see who's currently online in a server

---

### **Should Have** (Enhanced Experience)

**Rich Messaging** 14. As a user, I can edit my own messages so I can fix mistakes 15. As a user, I can delete my own messages so I can remove content 16. As a user, I can see when someone is typing in a channel 17. As a user, I can react to messages with emojis so I can respond quickly

**Server Management** 18. As a server owner, I can invite users via shareable link 19. As a server owner, I can delete channels I created 20. As a user, I can leave a server I'm no longer interested in

**User Experience** 21. As a user, I can upload an avatar so I can personalize my profile 22. As a user, I can see unread message indicators on channels 23. As a user, I can search for messages within a channel

---

### **Could Have** (Nice to Have)

**Voice/Video (WebRTC)** 24. As a user, I can join a voice channel so I can talk with others 25. As a user, I can see who's in a voice channel 26. As a user, I can mute/unmute my microphone 27. As a user, I can enable/disable video in a voice channel

**Advanced Features** 28. As a user, I can mention other users with @ so they get notified 29. As a user, I can attach images/files to messages 30. As a user, I can create threads from messages for focused discussions 31. As a server owner, I can assign roles to users (Admin, Moderator, Member) 32. As a user, I can receive notifications for mentions/DMs

**Polish** 33. As a user, I can use markdown formatting in messages (bold, italic, code blocks) 34. As a user, I can see link previews for URLs shared in chat

---

### **Won't Have** (Out of Scope)

35. Custom emoji uploads
36. Bot/webhook integrations
37. Screen sharing
38. Direct messages (DMs) between users
39. Message pinning
40. Advanced permission systems
41. Video streaming/Go Live features
42. Mobile apps (web-only for now)
